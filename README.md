# 闲鱼商品采集器 Pro

一个基于 Chrome 浏览器扩展插件的 闲鱼 商品采集工具，支持搜索结果采集、手动勾选采集、单链接采集、商品详情抓取、多格式导出，以及商品高清图 ZIP 打包。


## 版本说明

本项目目前只保留和维护 **Chrome 版本**。其他浏览器不作为当前项目的测试和维护目标。

## 主要功能

### 三种采集模式

- **全量采集**：输入关键词、页数和并发数，自动采集搜索结果。
- **手动勾选采集**：在搜索结果页注入复选框和底部工具栏，手动挑选商品后再采集。
- **单链接采集**：粘贴闲鱼商品链接或分享文案，自动提取链接并采集单个商品。

### 商品详情采集

支持采集以下字段：

- 序号
- 商品封面
- 商品链接
- 发布地
- 想要数
- 浏览量
- 价格
- 店铺名称
- 产品文案
- 查看图片

### 商品图片 ZIP 导出

勾选“采集商品图片”后，会额外导出 ZIP：

- ZIP 内包含数据文件和商品图片。
- 图片统一放在 `images/` 目录下。
- 每个商品按文案前若干字符建立独立文件夹。
- 商品图只采集详情页主轮播高清图。
- 过滤缩略图、头像、图标、标签图、站点 logo 和重复图片。

高清图采集策略：

- 只读取详情页主轮播区域：`[class*="carouselItem"] img`
- 排除 slick 轮播克隆图：`slick-cloned`
- 只保留高清图：`naturalWidth >= 700` 或 URL 包含 `_790x10000Q90`
- 按原图 ID 去重
- 不回退低清缩略图

### 多格式导出

- **CSV**：适合二次处理。
- **HTML**：支持图片预览、筛选、排序、文案弹窗查看。
- **Excel / XLSX**：支持封面图片锚定到单元格，避免图片越界。
- **ZIP**：数据文件 + 商品高清图。

### 自定义导出字段

扩展提供独立选项页，可勾选需要导出的字段。配置会保存在浏览器本地。

## 相比原项目的主要改动

### 体验优化

| 改动 | 说明 |

| 文案提取优化 | 使用更稳定的描述选择器，并通过 `innerText` 保留换行和符号 |

| 详情页图片提取 | 只采主轮播高清图，过滤低清和重复图 |
| 选项页保存方式 | 字段配置改为手动保存 |

### Bug 修复

- 修复 `xlsx-lite.js` 中异常引号导致的 Chrome 语法错误。
- 修复正则表达式中控制字符导致的解析错误。
- 修复 `sendProgress` 函数括号缺失导致的语法错误。
- 修复缺少 `storage` 权限导致配置保存失败的问题。

## 安装与使用

### 安装

方式一：使用 Git 下载

```bash
git clone https://github.com/xmz2018/xianyucaiji-pro.git
```

方式二：直接下载 ZIP

1. 打开项目页面。
2. 点击 `Code`。
3. 点击 `Download ZIP`。
4. 解压 ZIP 文件。

加载到 Chrome：

1. 打开 Chrome。
2. 地址栏输入 `chrome://extensions`。
3. 开启右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择项目文件夹。

注意：请选择包含 `manifest.json` 的那一层文件夹。如果是下载 ZIP 后解压，通常选择解压出来的 `xianyucaiji-pro-main` 或 `xianyucaiji-pro` 文件夹。

### 使用方法

### 全量采集

1. 打开闲鱼 / Goofish 页面。
2. 点击扩展图标。
3. 选择“全量采集”。
4. 输入关键词、采集页数和并发数。
5. 选择是否采集详情、是否采集商品图片、导出格式。
6. 点击“开始采集”。

### 手动勾选采集

1. 打开闲鱼搜索结果页。
2. 点击扩展图标。
3. 选择“手动勾选”。
4. 点击开始后，在页面中勾选需要采集的商品。
5. 点击页面底部工具栏中的确认采集。

### 单链接采集

1. 点击扩展图标。
2. 选择“单链接采集”。
3. 粘贴商品链接或闲鱼分享文案。
4. 选择导出格式和图片采集选项。
5. 开始采集。

## 权限说明

| 权限 | 用途 |
| --- | --- |
| `scripting` | 向闲鱼页面注入采集脚本 |
| `tabs` | 打开详情页并采集商品信息 |
| `storage` | 保存自定义导出字段配置 |
| `host_permissions` | 访问闲鱼页面和图片 CDN |

## 项目结构

```text
background.js   详情页采集、高清图提取、消息处理
content.js      搜索页采集、勾选模式、导出逻辑、ZIP 打包
popup.html      扩展弹窗页面
popup.js        弹窗交互、模式切换、任务启动
styles.css      弹窗与页面注入样式
options.html    字段配置页
options.js      字段配置逻辑
options.css     字段配置页样式
xlsx-lite.js    轻量 XLSX 与 ZIP 生成工具
manifest.json   浏览器扩展配置
```

## 开发调试

修改源码后，在浏览器扩展管理页点击“重新加载”，再刷新闲鱼页面。

建议调试顺序：

1. 先在 Console 确认没有语法错误。
2. 再测试普通 CSV 导出。
3. 再测试详情采集。
4. 最后测试 ZIP 图片导出。
